Cllr: Log-Likelihood Ratio Cost

0.00 = very well calibrated, 1.00 = no information

Measures how well-calibrated the likelihood ratios are. A perfect system scores 0, a system with no useful information scores 1.

EER: equal error rate

0% = perfect separation, 50% = no discriminative power

The Equal Error Rate (EER) is the point where the false acceptance rate and false rejection rate are equal. A lower value indicates better separation between authentic and non-authentic samples.

SD: Standard Deviation

small = consistent, large = variable

Standard deviation of Cllr across cross-validation folds. A low value indicates consistent performance across runs, a high value suggests the model is sensitive to the choice of data split.
